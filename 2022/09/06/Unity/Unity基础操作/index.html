入门Unity客户端开发

（1）

游戏引擎：

游戏开发工具箱（？） - 物理系统、UI系统、导航系统

虚幻引擎：

  • 图形能力超强

  • C++

  • 市场占有率相比unity低（主机的话高）

unity：

  • 多种渲染管线，图形能力也能很强

  • C#

  • 使用人数多，社区资源丰富

（2）

  • 窗口介绍

Hierachy： 层级面板，游戏场景中的资源，比如UI、模型

Scene： 场景面板，用于管理游戏场景中的各种游戏物体

Game： 游戏场景面板，实际的玩家视角

Project： 项目面板，用于管理项目中的所有资源

Inspedtor： 检查面板，即属性窗口，查看 Hierachy 或 Project 中组员的属性

  • 窗口布局

（3）

  • 游戏场景概念：

默认创建一个 SampleScene 场景文件

新建场景快捷键：Ctrl+N

保存场景快捷键：Ctrl+S

F2 改名

  • 游戏场景漫游：

  • 按住鼠标右：

    • 鼠标拖动旋转视角

    • WASD场景漫游

  • 鼠标中键：

    • 鼠标拖动平移视角

  • 按住alt：

    • 鼠标右键按住，拖动前后缩放
