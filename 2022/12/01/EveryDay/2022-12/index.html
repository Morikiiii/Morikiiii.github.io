MC 加资源包加光影流程：

  • 下载 JAVA （如果双击安装没用，就在对应文件夹 cmd 里 java -jar name.jar）

  • 下载安装对应版本的 forge - installer.jar【文件可以放在mods里】

  • 下载安装对应版本的 OptiFine.jar【同上】

  • 资源包放在 resourcepacks 里，zip 就行

  • 光影：没有 shaderpacks 就新建文件夹，同样 zip 就行【推荐BSL：BITSLABLAB - BSL Shaders】
